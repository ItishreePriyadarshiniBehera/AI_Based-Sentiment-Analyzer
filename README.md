# AI-Based Sentiment Analyzer

## Description
This project is an AI-Based Sentiment Analyzer developed using Python, Flask, Hugging Face Transformers, and MongoDB. It analyzes user-entered text and predicts whether the sentiment is Positive, Negative, or Neutral. 

---

## Features

- Analyze text sentiment
- Positive, Negative, and Neutral classification
- User-friendly web interface
- Real-time sentiment analysis

---

## Technologies Used

- Python
- Flask
- PyTorch
- HTML
- CSS

---



## How to Run

### 1. Install the required libraries

pip install ~r requirements.txt


### 2. Run the application

python app.py

### 3. Open in Browser


http://127.0.0.1:5000
